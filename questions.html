1. What is JavaScript. How to use it?

JavaScript is a programming language commonly used for web development. It allows us to add interactivity, dynamic content, and behavior to websites.
we can include JavaScript code directly within HTML using the script tag and we can also link to external JavaScript files using the src attribute of the script tag.

2. What is the use of isNaN function?

The Number.isNaN() static method determines whether the passed value is the number value NaN, and returns false if the input is not of the Number type. It is a more robust version of the original, global isNaN() function.

3. What is negative Infinity?

NEGATIVE_INFINITY is a special numeric value that is returned when an arithmetic operation or mathematical function generates a negative value greater than the largest representable number in JavaScript

7. What is the difference between ViewState and SessionState?

The basic difference between these two is that the ViewState is to manage state at the client's end, making state management easy for end-user while SessionState manages state at the server's end, making it easy to manage content from this end too. ViewState: It is maintained at only one level that is page-level.

8. What is === operator?

The strict equality operator === , also known as triple equals, compares both the value and the type of its operands.

11. What are all the looping structures in JavaScript?

JavaScript provides several looping structures for repetitive execution of code. The traditional for loop allows for precise control over initialization, condition, and iteration. The while loop repeats a block of code while a specified condition is true, and the do-while loop ensures at least one execution of the code block.

12. How can you convert the string of any base to an integer in JavaScript?

In JavaScript, we can convert a string representing a number in any base to an integer using the parseInt function. The parseInt function takes two arguments: the string to be converted, and the base of the numeral system. 

var binaryString = "101010";
var decimalNumber = parseInt(binaryString, 2);
console.log(decimalNumber); // Output: 42

13. What is the function of the delete operator?

In JavaScript, the delete operator is used to delete a property from an object. Its primary purpose is to remove a property along with its value from an object.

15. How can a page be forced to load another page in JavaScript?

To force a page to load another page in JavaScript, you can use the window.location object to change the current URL. There are different ways to achieve this, depending on our requirements. Here are a couple of common methods:

  • window.location.href
  • window.location.replace()

16. What are the disadvantages of using innerHTML in JavaScript?

While innerHTML is a convenient and widely used property in JavaScript for manipulating the content of HTML elements, it comes with some potential disadvantages:

  • Security Risks (Cross-Site Scripting - XSS)
  • Performance Impact
  • Overwrites Event Listeners and Data
  • Limited Error Handling
